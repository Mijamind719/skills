# skills

Mijamind's reusable Codex/agent skills.

## Available skills

| Skill | Description | Path |
| --- | --- | --- |
| llm-wiki | Personal knowledge base manager using LLMs to initialize, ingest, query, search, and maintain persistent wiki-style knowledge bases. | [`llm-wiki/SKILL.md`](llm-wiki/SKILL.md) |

## Usage

Install or copy a skill directory into your local Codex skills folder, then invoke it by name, for example:

```text
$llm-wiki initialize a project wiki
```
